Dit is het prototype voor een applicatie die incidenten kan registreren en managen.

Het is mogelijk om in dit prototype:
- Incidenten te melden
- Overzicht te bekijken
- Incidenten muteren
- Statestieken zien(niet met data uit de database)
- Gebruikers overzicht(niet met data uit de database)

In de folder database zijn 2 sql scripts te vinden:
witchidents.sql 	om alle tabellen en data te importeren. Je zult eerst een mysql database moeten aanmaken die "Witchidents" heet
user.sql 		om de database gebruiker toe te voegen die toegang heeft tot de database.

Dit prototype is ontwikkeld volgens het MVC model.
Er is gebruik gemaakt van bootstrap voor de opmaak.
Dit project is een git repository.
Er is vooral php5 gebruikt
Verder zijn javascript, HTML5 en MYSQL gebruikt
